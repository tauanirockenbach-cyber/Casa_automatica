# Casa_automatica
Casa automática inteligente para máximo conforto e segurança. Utiliza sensores de movimento (PIR) para iluminação, temperatura/umidade para climatização, gás (MQ-135) para detecção de vazamentos e nível de água para controle de caixas d'água. Ideal para automação residencial completa, monitoramento remoto e economia de energia com segurança total.
